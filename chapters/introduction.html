Definition

PWA: An application with simple HTML5/CSS3 and JS that can be installed to the homescreen of a device without needing the user to go through an app store

The Web App Manifest: The web app manifest is a simple JSON file in order to control how our application appears to the user

PWA launching properties

  • Unique icon and name
  • Displays a transitional screen to the user
  • Provides default display characteristics to the browser

The Web App Manifest orchestrate all that stuff
